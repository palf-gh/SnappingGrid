# SnappingGrid

## English

A Glyphs 3 plugin that displays a customizable snapping grid and snaps nodes to grid intersections on mouse release.

### Features

- **Main grid / Sub grid** display
- Grid mode: Division or Unit
- Individual H/V settings with sync option
- Customizable main/sub colors
- **Triangular (hexagonal) grid** — choose horizontal or vertical orientation
- Node snapping with handle tracking
- Italic shear support
- **Per-font settings** saved in the font file
- Localized UI: English, 日本語, 简体中文, 한국어

### Usage

- **View** menu → **Show Snapping Grid** to toggle visibility
- **Edit** menu → **Snapping Grid Settings…** to configure

### Settings

| Option | Description |
|---|---|
| Grid Shape | Square / Triangle |
| Orientation | Horizontal / Vertical (triangle only) |
| Grid Mode | Division / Unit |
| Main Grid | H/V spacing (Sync H/V available) |
| Sub Grid (main subdivisions) | Sub grid subdivisions |
| Main Color / Sub Color | Grid line colors |
| Enable snap | Toggle node snapping |
| Reset | Restore default values |

## 日本語

グリッドの表示とノードのスナップ機能を提供する Glyphs 3 プラグインです。

### 機能

- **メイングリッド / サブグリッド**の表示
- グリッド方式の切り替え（分割数 / Unit数）
- H/V 個別設定と縦横同期
- メイン色・サブ色のカスタマイズ
- **三角形グリッド（六角形グリッド）** ─ 水平・垂直方向を選択可能

- UI ローカライズ対応：English、日本語、简体中文、한국어

### 使い方

- **表示**メニュー → **スナッピンググリッドを表示** でグリッドの表示/非表示を切り替え
- **編集**メニュー → **スナッピンググリッドの設定…** で設定を変更

### 設定項目

| 項目 | 内容 |
|---|---|
| グリッド形状 | 方眼 / 三角形 |
| 方向 | 水平 / 垂直（三角形グリッドのみ） |
| グリッド方式 | 分割数 / Unit数 |
| メイングリッド | H/V 間隔（縦横同期あり） |
| サブグリッド（メインの分割数） | サブグリッドの分割数 |
| メイン色 / サブ色 | グリッド線の色 |
| スナップを有効にする | ノードの自動スナップ ON/OFF |
| リセット | 設定をデフォルト値に戻す |

## 简体中文

这是一款 Glyphs 3 插件，提供可自定义的网格显示和节点吸附功能。

### 功能

- **主网格 / 子网格**显示
- 网格方式切换（分割数 / Unit数）
- 水平/垂直独立设置及同步
- 主色/子色自定义
- **三角形网格（六边形网格）** ─ 可选水平或垂直方向
- 节点移动时自动吸附（含手柄联动）
- 支持意大利体倾斜角度
- **按字体文件独立保存设置**
- 界面语言：English、日本語、简体中文、한국어

### 使用方法

- **显示**菜单 → **显示吸附网格** 切换网格显示/隐藏
- **编辑**菜单 → **吸附网格设置…** 打开设置

### 设置项目

| 项目 | 说明 |
|---|---|
| 网格形状 | 方格 / 三角形 |
| 方向 | 水平 / 垂直（仅三角形网格） |
| 网格方式 | 分割数 / Unit数 |
| 主网格 | H/V 间距（可同步 H/V） |
| 子网格（主网格分割数） | 子网格分割数 |
| 主色 / 子色 | 网格线颜色 |
| 启用吸附 | 启用/禁用节点吸附 |
| 重置 | 恢复默认值 |

## 한국어

그리드 표시와 노드 스냅 기능을 제공하는 Glyphs 3 플러그인입니다.

### 기능

- **메인 그리드 / 서브 그리드** 표시
- 그리드 방식 전환（분할 수 / Unit 수）
- H/V 개별 설정 및 H/V 동기화
- 메인/서브 색상 커스터마이징
- **삼각형 그리드（육각형 그리드）** ─ 수평/수직 방향 선택 가능
- 노드 이동 시 자동 스냅（핸들 연동）
- 이탤릭 각도 대응 그리드 시어
- **폰트 파일별 설정 저장**
- UI 언어：English、日本語、简体中文、한국어

### 사용 방법

- **보기** 메뉴 → **스냅 그리드 표시** 로 그리드 표시/숨기기 전환
- **편집** 메뉴 → **스냅 그리드 설정…** 으로 설정 변경

### 설정 항목

| 항목 | 내용 |
|---|---|
| 그리드 형태 | 사각형 / 삼각형 |
| 방향 | 수평 / 수직（삼각형 그리드만） |
| 그리드 방식 | 분할 수 / Unit 수 |
| 메인 그리드 | H/V 간격（H/V 동기화 가능） |
| 서브 그리드（메인 분할 수） | 서브 그리드 분할 수 |
| 메인 색상 / 서브 색상 | 그리드 선 색상 |
| 스냅 활성화 | 노드 자동 스냅 ON/OFF |
| 초기화 | 기본값으로 초기화 |
